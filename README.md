## <img src="https://api.iconify.design/mdi/information-outline.svg?color=%236a5acd" width="24" height="24"/> Overview

Neuromo combines **computer vision AI** with the **Pomodoro Technique** to help you maintain focus and combat drowsiness during work sessions. Real-time facial landmark detection monitors your attention while respecting your privacy.

<!-- Feature Cards with Borders -->
## <img src="https://api.iconify.design/mdi/star-circle.svg?color=%23ffd700" width="24" height="24"/> Features

<table>
<tr>
<td width="33%" align="center">
<img src="https://api.iconify.design/mdi/brain.svg?color=%236a5acd" width="48" height="48"/>

### AI Monitoring

Real-time face detection using MediaPipe with 468 facial landmarks. Monitors drowsiness through Eye Aspect Ratio (EAR) and tracks focus via head pose analysis.

</td>
<td width="33%" align="center">
<img src="https://api.iconify.design/mdi/timer-outline.svg?color=%23ff6347" width="48" height="48"/>

### Pomodoro Timer

Auto-loop system with 25min Focus and 5min Break cycles. Privacy-first camera that turns OFF during breaks automatically.

</td>
<td width="33%" align="center">
<img src="https://api.iconify.design/mdi/music-note.svg?color=%2300bcd4" width="48" height="48"/>

### Immersive Mode

Integrated Lofi music player, dynamic tab countdown, desktop notifications, and safety features.

</td>
</tr>
</table>

## <img src="https://api.iconify.design/mdi/rocket-launch.svg?color=%23ff6347" width="24" height="24"/> Quick Start

```bash
# Clone the repository
git clone https://github.com/C0D3K0NG/Neuromo.git
cd Neuromo

# Install dependencies
pip install -r requirements.txt

# Launch the application
python app.py
```

<div align="center">

**Access at:** `http://127.0.0.1:8969`
